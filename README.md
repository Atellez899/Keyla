# Detalle minimalista

Sitio web estático en español con un diseño sobrio y elegante para compartir un detalle especial.

## Publicar en GitHub Pages

1. Sube este repositorio a GitHub.
2. En GitHub, ve a **Settings → Pages**.
3. En **Branch**, selecciona `main` y la carpeta raíz (`/root`).
4. Guarda los cambios. GitHub mostrará la URL publicada después de unos segundos.

### Uso local

Abre `index.html` en tu navegador o usa un servidor local sencillo si prefieres.

```
# Ejemplo con Python
python3 -m http.server 8000
```

Luego visita `http://localhost:8000`.
